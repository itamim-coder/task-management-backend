## Tech Stack 🛠️

- **Frontend:** Next.js, TypeScript, Redux  
- **Backend:** Node.js, Express.js, Mongoose  
- **Database:** MongoDB, 
- **Authentication:** JWT  

## Installation ⚙️

Clone the repo and install dependencies:

```bash
git clone https://github.com/itamim-coder/event-ease-backend.git
cd event-ease-backend
npm install
```

Run the application:

```bash
nodemon start-dev
```
